# MPL ID S17 Analytics Dashboard

Big Data analytics project for MPL Indonesia Season 17 using PostgreSQL, Metabase, and sentiment analysis on YouTube comments.

## Features

- MPL S17 Standings Analysis
- Hero Meta Analysis
- Pick & Ban Statistics
- YouTube Comment Sentiment Analysis
- Interactive Metabase Dashboard

## Tech Stack

### Data Collection
- Playwright
- YouTube Data API v3

### Data Processing
- Python
- Pandas

### AI & NLP
- Groq LLM

### Database
- PostgreSQL

### Workflow Automation
- n8n

### Data Visualization
- Metabase
